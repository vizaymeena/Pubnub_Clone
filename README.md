# Pubnub UI Clone

this project is a **UI only clone** of the PubNub website, built using **React.js**.  
The goal of this project is to practice and demonstrate **frontend layout structuring, component design, and responsive UI patterns** commonly used in current industry standards

# Features
- Responsiveness for different devices
- Clean Strucutre 
- Maintainable css
- Destop First Layout
- Dropdown Sections 

# TechStack
- React
- Javascript
- Lucide-Icons
- Css
